This folder is intended to hold representative images used in the project README.

Place a few sample face images here so the README can reference them. Example filenames used in the README:
- F100000.jpg
- F100010.jpg
- F100020.jpg

You can copy these from `backend/data/face_images/`:
Example PowerShell command:
Copy-Item -Path .\backend\data\face_images\F100000.jpg -Destination .\docs\images\F100000.jpg
